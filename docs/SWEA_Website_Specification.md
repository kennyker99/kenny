| 项目 | 说明 |
|------|------|

## 🔧 核心功能模块

### 1. 课程管理系统
```javascript
// 数据结构
{
  modules: [
    {
      id: "mi1",
      name: "汇市趋势",
      nameEn: "Fundamental Analysis",
      description: "理解宏观经济如何驱动市场",
      icon: "📊",
      color: "#FF6B6B",
      lessons: [
        {
          id: "mi1-1",
          title: "经济缸中的游鱼",
          duration: 763, // 秒
          videoUrl: "...",
          content: "...",
          keyPoints: ["通胀", "货币政策"],
          difficulty: 2,
          order: 1
        }
      ]
    }
  ]
}
```

### 2. 用户进度追踪
```javascript
{
  userId: "user123",
  progress: {
    overall: 0.40, // 40%
    modules: {
      mi1: { completed: true, progress: 0.60 },
      mi2: { completed: false, progress: 0.40 }
    },
    lessons: {
      "mi1-1": { completed: true, watchedAt: "2026-06-15" },
      "mi1-2": { completed: false, progress: 0.30 }
    }
  }
}
```

### 3. 交互工具系统
```javascript
{
  tools: [
    {
      id: "risk-calculator",
      name: "风险计算器",
      category: "risk-management",
      inputs: [
        { name: "accountSize", label: "账户金额", type: "number" },
        { name: "riskPercent", label: "风险百分比", type: "number" }
      ],
      calculate: (inputs) => {
        return inputs.accountSize * inputs.riskPercent / 100;
      }
    }
  ]
}
```

## 🎨 设计系统

### 色彩方案
```
主色: #FF6B6B (红色 - 能量、行动)
辅色: #4ECDC4 (青色 - 信任、专业)
强调色: #FFE66D (黄色 - 警告、重点)
背景: #F7F9FC (浅灰 - 清爽)
文字: #2C3E50 (深灰 - 易读)

```

副标题: Inter SemiBold (18-24px)
